ECU = экю
1 ECU = 1g gold

Наименования:
    FS = Фунт стерлингов
    IF = Ирланский фунт
    KR = Золотая крона
    PR = Португальский реал
    AH = Ахче
    LI = Ливр

    IR = Испанский реал
    PE = Песо
    
    GU = Гюльден
    cKC = Крейцер
    cPH = пфенинг
    
    FR = Швейцарский Франк

Конвертация волют:
    1 IF = n ECU
    1 PR = n ECU
    1 KR = 2 ECU
    1 LI = 0.1 ECU
    1 AH = 0.082 ECU

    1 FS = 20 шилингов = 240 пени = 24 ECU
    1 шилинг = 1.2 ECU
    1 пени = 0.1 ECU

    1 GU = 60 KC = 240 PH = 1.5 ECU
    1 сKC = 0.025 ECU
    1 сPH = 0.0625 ECU

    1 IR = 25 PE = 1.2 ECU
    1 PE = 0.048 ECU

    1 FR = 100 FR = 0.1 ECU
    1 cFR = 0.0001 ECU
